Opens socket for CAN bus communication via socketcan. Returns socket identifier which is used as an argument for future can::read(), can::write() and can::close() calls.

Parameters
fdOpen as normal CAN or CANFD.
channelNetwork channel to use, e.g. can0 or vcan0.
Returns
const int Socket identifier.
-1 If socket creation failed.
-2 If could not enable CANFD.
-3 If could not bind socket.
