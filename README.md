Instructions to run the code

1) Clone the repository :-
git clone https://github.com/joshir14/7166CEM_Coursework.git

2) Change directory :-
cd 7166CEM_Coursework

3) Compile the code :-
cmake -DCMAKE_EXPORT_COMPILE_COMMANDS=YES

4) Build the code :-
make

4) Change directory :-
cd bin

5) Run Applcation :-
./Application
